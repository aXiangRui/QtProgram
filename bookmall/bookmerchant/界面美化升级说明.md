## 🔧 技术实现

### 渐变语法
```css
qlineargradient(x1:0, y1:0, x2:1, y2:0,
    stop:0 颜色1,
    stop:0.5 颜色2,
    stop:1 颜色3)
```

### 半透明语法
```css
rgba(红, 绿, 蓝, 透明度)
rgba(255, 255, 255, 0.95) /* 95%白色 */
```

### 圆角语法
```css
border-radius: 12px;           /* 全部圆角 */
border-top-left-radius: 12px;  /* 单独圆角 */
```
